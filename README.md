# Nyratoolbox
个人常用命令行工具箱。

收入一些轻量级、可直接运行的小工具，主要面向日常测试、数据处理、格式验证、脚本实验和临时自动化和创建字典。

工具多为：
- 单文件或少依赖
- 命令行直接运行
- 参数清晰
- 便于复制、修改和二次封装

> 本仓库中的工具仅用于合法的软件测试、教学演示、数据格式验证和个人自动化用途。  
> 请勿将任何工具用于撞库、垃圾邮件、钓鱼、批量注册、骚扰、绕过风控或其他非法 / 不当用途。

---

## 目录结构

```text
Nyratoolbox/
├── number_dictionary       # 生成 QQ号与来自真实号段的随机手机号 以及 手机号/QQ号格式邮箱
├── SFZ_dictionary          # 生成 格式合法的 大陆身份证号码
├── ShortID_dictionary      # 生成指定长度的十六进制 ShortID 穷举列表
├── Subnetting              # IPv4 / IPv6 子网计算、地址规划、ACL wildcard 和 DHCP 地址池工具
└── README.md               # 项目根目录说明
```
